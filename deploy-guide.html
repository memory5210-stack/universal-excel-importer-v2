🚀 万能导入 V2 - Vercel 部署指南

5 分钟完成部署，即刻访问您的在线应用

1 创建 Neon 数据库
  1. 访问 neon.tech 并登录
  2. 点击 "New Project"
  3. 填写项目名称：universal-import-db
  4. 点击 "Create"
  5. 在 Connection Details 区域复制连接字符串（包含 ?sslmode=require）
postgresql://user:password@ep-xxx.region.aws.neon.tech/neondb?sslmode=require
打开 Neon
✅ 完成后保存连接字符串，稍后需要填入 Vercel
2 获取 DeepSeek API Key
  1. 访问 DeepSeek 开放平台
  2. 登录后进入 "API Keys" 页面
  3. 点击 "Create API Key"
  4. 复制生成的 Key（以 sk- 开头）
打开 DeepSeek
3 部署到 Vercel
  1. 访问 vercel.com/new
  2. 登录 Vercel（支持 GitHub 账号）
  3. 点击 "Import Git Repository"
  4. 搜索并选择 universal-excel-importer-v2
  5. 点击 "Import" 导入仓库
开始部署
💡 部署约需 2-3 分钟
4 配置环境变量

在 "Configure Project" 页面点击 "Add Environment Variable" 添加：

Name Value
DATABASE_URL 步骤 1 复制的 Neon 连接字符串
DEEPSEEK_API_KEY 步骤 2 获取的 API Key

然后点击 "Deploy"
